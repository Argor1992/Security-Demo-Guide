Starten unter Windows

Unter Windows müssen Sie lediglich den Ordner target öffnen und die Datei company-portal-1.1-SNAPSHOT.jar doppelklicken. Bitte haben Sie etwas Geduld, bis das Programm gestartet ist, was im schlimmsten Fall ~30 Sekunden dauern kann. Das Setzen einer Umgebungsvariable ist für das Starten der Demo nicht notwendig, wird aber benötigt, wenn Sie das Programm über die Konsole starten möchten.

Sollte das Programm nicht starten, stellen Sie bitte sicher, dass sie mindestens Java 11 installiert haben. Java 11, 14, 15 und Java 17 wurden unter Windows 10 und 11 getestet und haben funktioniert.
